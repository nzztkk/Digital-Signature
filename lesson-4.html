Шаг 1. Генерация ключей

(pk, sk) <- KeyGen()
pk — public key (публичный), sk — secret key (приватный).

Шаг 2. Считаем хеш сообщения

h = H(m)
m — сообщение, h — короткий отпечаток этого сообщения.

Шаг 3. Создаем подпись

s = Sign(sk, h)
Приватный ключ «привязывает» подпись к хешу. Без приватного ключа создать такую же подпись нельзя.

Шаг 4. Проверяем подпись

valid = Verify(pk, h, s)
Если результат true (истина), значит подпись настоящая и текст не изменился.

Немного про RSA (упрощенно)

n = p * q
e * d mod phi(n) = 1
s = h^d mod n, проверка: h ?= s^e mod n
Это уже взрослая математика. Тебе важно запомнить идею: один ключ подписывает, второй проверяет.
