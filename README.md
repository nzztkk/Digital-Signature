# Crypto Pipeline Platform

Интерактивная платформа для студентов по цифровой подписи, SHA-256 и асимметричному шифрованию.

## Что реализовано

- Модуль архитектурного ландшафта (ключи, хэш, подпись, проверка)
- Таблица сравнения RSA и ECDSA
- Memory view SHA-256: ячейки, блоки, schedule, раунды, digest
- Data flow pipeline: raw -> hashed -> signed -> transmitted -> verified
- Практикум: RSA-PSS подпись/проверка + AES-GCM + RSA-OAEP
- Формулы в LaTeX (MathJax)

## Основные файлы

- `index.html` — вся платформа (single-page)
- `styles.css` — оформление и визуальные блоки
- `script.js` — криптографическая логика и анимации

## Запуск локально

Открой файл `index.html` в браузере двойным кликом.

## Публикация на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него файлы проекта (`index.html`, `styles.css`, `script.js`, `README.md`).
3. На GitHub открой `Settings` -> `Pages`.
4. В разделе `Build and deployment` выбери:
   - `Source: Deploy from a branch`
   - `Branch: main` и `/ (root)`
5. Нажми `Save`.
6. Подожди 1-2 минуты и открой ссылку вида:
   `https://<твой-логин>.github.io/<имя-репозитория>/`

## Важное замечание

Это учебный проект. Для реальных юридически значимых подписей нужны:

- сертификаты и инфраструктура доверия (PKI),
- соблюдение законов и стандартов,
- защищенное хранение приватного ключа.
